# Docs Structure

This directory is used for GitHub Pages deployment.

- `index.html`: project page entrypoint
- `assets/`: frontend assets
  - `css/`: Bulma/vendor styles and custom styles
  - `js/`: interaction scripts
  - `images/`: figures used in the project page
- `paper/`: paper source files and reference figures

Keep `index.html` paths relative to `docs/` so the site works on GitHub Pages.
